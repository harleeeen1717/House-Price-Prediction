## Project Overview

This project predicts house prices using Machine Learning algorithms. The dataset was obtained from Kaggle and contains various features such as area, number of bedrooms, bathrooms, parking, air conditioning, furnishing status, and more.

The project includes data preprocessing, visualization, model training, evaluation, and house price prediction.

## Conclusion

This project demonstrates how machine learning can be used to predict house prices based on various housing features. The dataset was preprocessed, categorical variables were encoded, and two regression models were trained and evaluated.

Among the two models, Linear Regression achieved a higher R² score of **0.6495**, while Random Forest Regression achieved an R² score of **0.6115** on the test data. Based on these results, Linear Regression performed better for this dataset.

This project helped in understanding the complete machine learning workflow, including data preprocessing, visualization, model training, evaluation, and prediction.
